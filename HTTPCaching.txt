HTTP memiliki fitur yang bernama caching
Caching adalah menyimpan data di client sampai batas waktu yang sudah ditentukan,
sehingga jika client ingin melakukan request resource yang sama,cukup ambil reource 
nya di client,tanpa harus meminta ulang ke server
HTTP Chacing sangat cocok untuk reource file static yang jarang di ubah-ubah
seperti gambar,video,audio,dll.
Cache-Control berisi informasi seperti berapa lama client bisa menyimpan data
response tersebut,sehingga tidak perlu meminta ulang ke server
